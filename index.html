Overview:
This project presents a data-driven computational framework for calibrating and validating heterogeneous agent-based pedestrian models using real-world multi-agent trajectory data. By integrating Particle Swarm Optimization (PSO) with a deep learning surrogate model (TabNet), the framework accelerates parameter calibration while maintaining accuracy in reproducing pedestrian behaviors in various environments.

Figure: Snapshots of pedestrian simulation at different steps.

üîç Output & Results

The framework enables:

  • Efficient calibration of multiple agent parameters from real-world data.
  • Simulation outputs that closely match actual pedestrian velocities and trajectories.
  • Scalability for testing different environment layouts and crowd densities.
Validation results showed strong alignment between simulated and observed pedestrian flow patterns, reducing calibration runtime significantly compared to traditional brute-force methods.
